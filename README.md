ClalisSampleForJava
===================

ClalisをC#から使用するための簡単なサンプルです。  
ソースを読んで頂けると、APIの叩き方が分かるかと思います。  
APIはSOAP、JSON、XMLのインターフェースを用意しています。  
お好みのものをご使用ください。  
C#の場合はSOAPがおすすめです。  
下の方で、各画面とその使い方を簡単に説明しています。  
ソースやサンプルのプロジェクトを開くにはVisualStudio2010が必要です。  
また、サンプルプログラムの実行には.netFramwwork4.0が必要です。
